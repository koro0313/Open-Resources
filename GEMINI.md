# Gemini Rules

Please adhere to the following rules when working on this project:

1. **Do not try to test or verify features yourself**: The user will test and verify the features and provide the review. Do not run commands or write scripts to verify code correctness unless explicitly requested.
2. **Always use available skills**: Whenever performing tasks, check and use the available skills located in the `.agents/skills` directory first (e.g., `supabase`, `frontend-design`, `shadcn`, etc.).
3. **Always follow the docs**: Reference and follow the documentation located in the `docs` directory.
4. **Never run development or preview commands**: Do not run commands like `pnpm dev`, `pnpm preview`, or similar commands, as they can affect the user's workflow.
